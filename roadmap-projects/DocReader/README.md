# 📄 DocReader

문서 리더 앱 통합 샘플 프로젝트입니다.

## 사용 프레임워크

| 프레임워크 | 용도 |
|-----------|------|
| **SwiftUI** | 선언적 UI |
| **PDFKit** | PDF 뷰어/편집 |
| **Vision** | OCR 텍스트 인식 |
| **AVFoundation** | 문서 스캐너 |

## 주요 기능

- 📄 PDF 뷰어 (PDFKit)
- 📝 텍스트 추출 (Vision OCR)
- 📷 문서 스캔 (AVFoundation + VNDocumentCameraViewController)
- 🔍 텍스트 검색

## 학습 포인트

1. **PDFKit**: PDFView, PDFDocument
2. **Vision**: VNRecognizeTextRequest
3. **VisionKit**: VNDocumentCameraViewController
4. **AVFoundation**: 카메라 미리보기
