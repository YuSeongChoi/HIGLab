🗣️ Siri 음성 제어 앱 만들기

⭐ 난이도: ⭐⭐⭐ ⏱️ 예상 시간: 2-3h 📂 App Frameworks

App Intents로 "할 일 추가해줘"라고 말하면 동작하는 앱을 만듭니다.

✨ App Intents란?

App Intents는 앱의 기능을 Siri, 단축어, Spotlight에 노출합니다. iOS 16+에서 사용 가능하며, 이전의 SiriKit보다 훨씬 간단합니다.

📝 첫 번째 Intent 만들기

AddTodoIntent.swift
import AppIntents

struct AddTodoIntent: AppIntent {
    static var title: LocalizedStringResource = "할 일 추가"
    
    @Parameter(title: "제목")
    var todoTitle: String
    
    func perform() async throws -> some IntentResult {
        TodoStore.shared.add(title: todoTitle)
        return .result(dialog: "\(todoTitle) 추가 완료!")
    }
}

🎤 Siri 음성 명령 등록

Shortcuts.swift
struct TodoShortcuts: AppShortcutsProvider {
    static var appShortcuts: [AppShortcut] {
        AppShortcut(
            intent: AddTodoIntent(),
            phrases: [
                "할 일 추가해줘 \(.applicationName)",
                "\(.applicationName)에 할 일 추가"
            ],
            shortTitle: "할 일 추가",
            systemImageName: "plus.circle"
        )
    }
}

💡 HIG 팁

자연스러운 문장으로 phrases를 작성하세요. "할 일 추가해줘", "새 할 일 만들어줘" 처럼 사용자가 실제로 말할 법한 표현을 사용합니다.

🔄 파라미터 대화 흐름

파라미터가 없으면 Siri가 자동으로 물어봅니다:

Dialog.swift
guard let title = todoTitle else {
    throw $todoTitle.needsValueError("어떤 할 일을 추가할까요?")
}

📋 Entity 정의

Todo 항목을 Entity로 정의하면 Siri가 할 일을 제안하고 검색할 수 있습니다:

TodoEntity.swift
struct TodoEntity: AppEntity {
    static var typeDisplayRepresentation: TypeDisplayRepresentation {
        "할 일"
    }

    var id: UUID
    var title: String
    var isCompleted: Bool

    var displayRepresentation: DisplayRepresentation {
        DisplayRepresentation(title: "\(title)")
    }

    // Spotlight 검색 지원
    static var defaultQuery = TodoEntityQuery()
}

🔍 EntityQuery 구현

TodoEntityQuery.swift
struct TodoEntityQuery: EntityQuery {
    func entities(for identifiers: [UUID]) async throws -> [TodoEntity] {
        TodoStore.shared.todos.filter { identifiers.contains($0.id) }
    }

    func suggestedEntities() async throws -> [TodoEntity] {
        // 미완료 항목 3개 제안
        TodoStore.shared.todos
            .filter { !$0.isCompleted }
            .prefix(3)
            .map(TodoEntity.init)
    }
}

📱 SwiftUI 통합

AppIntentsUI를 사용해 Siri 응답을 SwiftUI로 꾸밀 수 있습니다:

AddTodoIntent+UI.swift
import AppIntentsUI

extension AddTodoIntent {
    @MainActor
    func perform() async throws -> some IntentResult & ProvidesDialog & ShowsSnippetView {
        let todo = TodoStore.shared.add(title: todoTitle)

        return .result(
            dialog: "\(todoTitle) 추가했어요!",
            view: TodoSnippetView(todo: todo)
        )
    }
}

struct TodoSnippetView: View {
    let todo: TodoEntity

    var body: some View {
        HStack {
            Image(systemName: "checkmark.circle")
            Text(todo.title)
        }
        .padding()
    }
}
