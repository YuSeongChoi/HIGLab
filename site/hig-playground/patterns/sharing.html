✓ 이렇게 하세요

  • 시스템 공유 시트(ShareLink / UIActivityViewController) 사용
  • 콘텐츠 타입에 맞는 미리보기 제공
  • 커스텀 액션은 최소한으로 유지
  • 이미지 공유 시 파일 크기 정보 표시

✗ 이렇게 하지 마세요

  • 커스텀 공유 UI를 직접 만들지 마세요
  • 시스템 공유 옵션을 제거하지 마세요
  • 커스텀 액션을 5개 이상 추가하지 마세요
  • URL 공유 시 미리보기를 생략하지 마세요

iPhone 미리보기에서 나쁜 패턴을 직접 확인해보세요

The system share sheet (UIActivityViewController) provides a consistent, familiar experience. It automatically adapts to content type, suggests relevant apps, and supports AirDrop. Always prefer the system share sheet over custom implementations.
