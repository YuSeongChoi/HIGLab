📚 HIG Guidelines — 직접 체험하기
✓ Do — 올바른 예시
✗ Don't — 나쁜 예시

✓ 이렇게 하세요

  • 모든 멀티태스킹 모드를 지원하세요
  • 사용 가능한 너비에 맞게 레이아웃을 적응시키세요
  • 크기 변경 시 상태를 유지하세요
  • Compact 너비에서 읽기 좋은 여백을 사용하세요

✗ 이렇게 하지 마세요

  • Full Screen 프레젠테이션을 가정하지 마세요
  • 고정된 너비를 하드코딩하지 마세요
  • Size Class 변경을 무시하지 마세요
  • Split View에서 필수 컨트롤을 숨기지 마세요

iPhone 미리보기에서 나쁜 패턴을 직접 확인해보세요

iPad supports multiple multitasking modes: Full Screen for single-app focus, Split View for side-by-side apps with adjustable ratios, and Slide Over for a floating compact panel. Apps should adapt their layout using size classes and NavigationSplitView.
