✓ 이렇게 하세요

  • 다양한 실행 취소 방법 제공 (흔들기, 버튼, 제스처)
  • 다단계 실행 취소로 충분한 히스토리 제공
  • 파괴적 작업 전 확인 대화상자 표시
  • 실행 취소와 함께 다시 실행도 제공

✗ 이렇게 하지 마세요

  • 실행 취소 없이 데이터 삭제
  • 단일 단계만 되돌리기 허용
  • 실행 취소만 제공하고 다시 실행 미제공
  • 확인 없이 되돌릴 수 없는 작업 실행
