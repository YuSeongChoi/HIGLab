Charting Data

차트 데이터를 효과적으로 표현하는 패턴을 직접 체험해보세요. 차트 유형, 데이터 포인트, 라벨, 범례 등을 조작하면 iPhone 화면에 즉시 반영됩니다.

✓ Do — 올바른 예시
✗ Don't — 나쁜 예시

✓ 이렇게 하세요

  • 데이터 포인트를 5~7개 이내로 유지
  • Pie 차트는 5개 이하 항목에 사용
  • 항상 라벨과 범례를 포함
  • 시스템 색상을 활용하여 접근성 확보
  • 애니메이션으로 데이터 변화를 표현
  • 차트 유형에 맞는 색상 대비 사용

✗ 이렇게 하지 마세요

  • ❌ 10개 이상의 데이터 포인트로 차트 과밀화
  • ❌ Pie 차트에 7개 이상 항목 사용
  • ❌ 라벨이나 범례 없이 차트 표시
  • ❌ Pie 차트에 단색 계열 색상 사용
  • ❌ 애니메이션 없이 데이터 전환

👆 컨트롤을 조작하여 나쁜 패턴을 직접 확인해보세요

Apple의 Swift Charts 프레임워크는 데이터를 명확하고 아름답게 시각화합니다. 차트는 데이터의 의미를 전달하는 것이 핵심이며, 과도한 정보는 오히려 가독성을 해칩니다.
