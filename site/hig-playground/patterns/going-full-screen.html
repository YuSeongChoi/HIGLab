✓ 이렇게 하세요

  • 명확한 전체 화면 진입/종료 방법 제공
  • 컨트롤 오버레이로 주요 기능 접근 보장
  • 가로/세로 모드 모두 지원
  • 상태 바 숨김 시에도 시간 확인 수단 제공

✗ 이렇게 하지 마세요

  • 종료 방법 없이 전체 화면 강제 진입
  • 사용자 동의 없이 자동 전체 화면
  • 상태 바와 컨트롤 모두 숨기기
  • 화면 회전 잠금 강제 적용

전체 화면 모드는 몰입감을 높이지만, 사용자가 항상 제어할 수 있어야 합니다. 명확한 종료 방법과 필수 정보 접근을 보장하세요.
