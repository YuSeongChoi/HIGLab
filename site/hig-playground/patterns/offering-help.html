📚 HIG Guidelines — 직접 체험하기
✓ Do — 올바른 예시
✗ Don't — 나쁜 예시

✓ 이렇게 하세요

  • 관련 기능 근처에 맥락에 맞는 도움말을 표시하세요
  • 사용자가 쉽게 닫을 수 있도록 해주세요
  • 적절한 타이밍에 팁을 보여주세요
  • 점진적 공개를 위해 TipKit을 활용하세요

✗ 이렇게 하지 마세요

  • 매 화면 실행마다 도움말을 표시하지 마세요
  • 도움말 오버레이로 사용자 인터랙션을 차단하지 마세요
  • 도움말 텍스트에 전문 용어를 사용하지 마세요
  • 닫은 후에도 같은 팁을 반복 표시하지 마세요
