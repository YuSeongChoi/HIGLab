📚 HIG Guidelines — 직접 체험하기
✓ Do — 올바른 예시
✗ Don't — 나쁜 예시

✓ 이렇게 하세요

  • 검색 바에 명확한 플레이스홀더 제공
  • 자동 완성 제안 표시
  • Scope Bar로 결과 필터링 지원
  • 최근 검색어 저장 및 표시
  • 결과 없을 때 도움말 제공

✗ 이렇게 하지 마세요

  • 검색 접근성 숨기기 (스크롤해야 보이게)
  • 검색 중 화면 전체 차단
  • 결과 없을 때 빈 화면만 표시
  • 검색 취소 버튼 미제공

iPhone 미리보기에서 나쁜 패턴을 직접 확인해보세요
