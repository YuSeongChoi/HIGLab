📚 HIG Guidelines

✓ Do

  • 시스템 컬러를 사용하여 자동 다크 모드 대응
  • semantic color (label, secondaryLabel 등) 활용
  • 색상 외 다른 수단(아이콘, 텍스트)으로도 정보 전달
  • 충분한 색상 대비 확보 (4.5:1 이상)

✗ Don't

  • 하드코딩된 RGB 값으로 텍스트 색상 지정하지 마세요
  • 색상만으로 상태를 구분하지 마세요
  • 다크 모드에서 순수 흰색(#FFFFFF) 배경을 사용하지 마세요
  • 시스템 컬러와 충돌하는 커스텀 컬러를 쓰지 마세요

iOS 시스템 컬러는 라이트/다크 모드에 자동으로 적응합니다. semantic color를 사용하면 모든 외형 모드에서 적절한 색상이 적용됩니다.
