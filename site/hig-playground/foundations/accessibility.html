Accessibility

iOS 접근성 설정이 앱 UI에 미치는 영향을 직접 확인해보세요. Dynamic Type, Bold Text, Reduce Motion 등을 컨트롤 패널에서 변경하면 iPhone 화면에 즉시 반영됩니다.

●●●● WiFi 🔋
📚 HIG Guidelines

✓ Do

  • Dynamic Type을 지원하여 사용자가 원하는 텍스트 크기 사용
  • 모든 인터랙티브 요소에 접근성 레이블 제공
  • 최소 44pt 탭 영역 확보
  • 색상만으로 정보를 전달하지 않기
  • Reduce Motion 설정 시 애니메이션 대체

✗ Don't

  • 고정 폰트 크기를 사용하지 마세요
  • 접근성 레이블을 생략하지 마세요
  • 낮은 대비의 텍스트를 사용하지 마세요
  • Reduce Motion을 무시하지 마세요
  • 자동 재생 애니메이션을 제어 불가로 두지 마세요

iOS의 접근성 기능은 모든 사용자가 앱을 사용할 수 있게 합니다. Dynamic Type은 텍스트 크기를, Bold Text는 가독성을, Reduce Motion은 모션에 민감한 사용자를 지원합니다.
