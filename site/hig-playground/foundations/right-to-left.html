📚 HIG Guidelines — 직접 체험하기
✓ Do — 올바른 예시
✗ Don't — 나쁜 예시

✓ 이렇게 하세요

  • leading/trailing을 left/right 대신 사용하세요
  • 방향성 아이콘(화살표, 셰브론)은 RTL에서 미러링하세요
  • 실제 RTL 콘텐츠(아랍어, 히브리어)로 테스트하세요
  • Auto Layout / SwiftUI를 사용하면 자동으로 플립됩니다

✗ 이렇게 하지 마세요

  • left/right 마진을 하드코딩하지 마세요
  • 비대칭 아이콘의 미러링을 빠뜨리지 마세요
  • 체크마크, 플러스 같은 비방향성 아이콘은 미러링하지 마세요
  • 텍스트가 항상 왼쪽에서 오른쪽으로 흐른다고 가정하지 마세요

iPhone 미리보기에서 나쁜 패턴을 직접 확인해보세요

SwiftUI는 .environment(\.layoutDirection, .rightToLeft)로 RTL 레이아웃을 자동 지원합니다. leading/trailing 기반의 레이아웃을 사용하면 별도 코드 변경 없이 RTL을 지원할 수 있습니다. 방향성 아이콘(chevron, arrow)은 RTL에서 자동으로 미러링됩니다.
