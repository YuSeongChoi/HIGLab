Dark Mode

라이트/다크 모드 전환이 UI에 미치는 영향을 확인해보세요. 네비게이션 바, 카드, 리스트, 탭 바 등 모든 요소가 실시간으로 전환됩니다.

9:41
●●●● WiFi 🔋
📚 HIG Guidelines

✓ Do

  • semantic/adaptive 컬러 사용 (Color.primary 등)
  • 다크 모드에서 elevation으로 깊이 표현
  • 두 모드에서 모두 테스트하기
  • 시스템 머티리얼을 블러 배경에 활용

✗ Don't

  • 다크 모드에서 순수 검정(#000) 카드 배경 사용 금지
  • 하드코딩된 색상 값을 사용하지 마세요
  • 다크 모드에서 그림자 대신 elevation 사용
  • 라이트 전용 이미지를 다크에서 그대로 쓰지 마세요

다크 모드는 어두운 환경에서 눈의 피로를 줄이고 OLED 디스플레이에서 배터리를 절약합니다. iOS는 base와 elevated 두 수준의 배경을 제공하여 깊이를 표현합니다.
