📚 HIG Guidelines

✓ Do

  • Use Filled style for the primary action
  • Use verbs or short verb phrases as labels
  • Use title case for button titles
  • Limit button text to one or two words
  • Use system-provided button styles

✗ Don't

  • Don't use multiple Filled buttons side by side
  • Don't use overly long button labels
  • Don't use a button when a link is more appropriate
  • Don't use destructive style for non-destructive actions

iOS provides four button styles: Filled (borderedProminent) for primary actions, Tinted (bordered with tint) for secondary actions, Bordered for tertiary actions, and Plain (borderless) for minimal emphasis. Always use the highest emphasis button for the most important action.
