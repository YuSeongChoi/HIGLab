✓ Do — 올바른 예시
✗ Don't — 나쁜 예시

✓ 이렇게 하세요

  • Provide visual cues that content is scrollable (partial content visible at edges)
  • Use paging behavior for full-width items like onboarding or photo galleries
  • Show scroll indicators during active scrolling
  • Support both drag and swipe gestures
  • Use horizontal scrolling for related, browsable content

✗ 이렇게 하지 마세요

  • Don't nest scroll views in the same direction
  • Don't hide all scrollable content below the fold with no visual hint
  • Don't disable bounce unless you have a strong reason
  • Don't use horizontal scrolling for primary navigation lists
