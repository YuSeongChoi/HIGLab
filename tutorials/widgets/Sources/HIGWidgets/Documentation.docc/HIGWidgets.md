## Overview

5단계 성장고리 커리큘럼으로 WidgetKit의 핵심을 마스터합니다.
HIG 가이드라인을 코드로 직접 구현하며 배웁니다.

- **Ring 1**: HIG 위젯 가이드라인 이해
- **Ring 2**: Widget Extension 기본 구조
- **Ring 3**: HIG 준수 UI 디자인
- **Ring 4**: 데이터 연동 & 인터랙션
- **Ring 5**: 설정 & 퍼스널라이즈
