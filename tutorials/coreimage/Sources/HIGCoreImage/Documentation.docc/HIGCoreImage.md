# ``HIGCoreImage``

CoreImage 프레임워크를 활용한 이미지 필터링 및 처리 가이드입니다.

## Overview

이 튜토리얼에서는 Apple의 CoreImage 프레임워크를 사용하여 
강력한 이미지 필터링과 처리 기능을 구현하는 방법을 단계별로 배웁니다.

CoreImage는 iOS와 macOS에서 GPU 가속을 활용한 
고성능 이미지 처리를 제공하는 프레임워크입니다.

### 학습 내용

- CIImage와 CIContext의 역할 이해
- 200개 이상의 빌트인 필터 활용
- 필터 체이닝으로 복잡한 효과 구현
- CIDetector를 활용한 얼굴 감지
- 커스텀 CIKernel 작성
- Metal을 활용한 고성능 필터
- 실시간 카메라 필터 구현
- RAW 이미지 처리
- 메모리와 성능 최적화
