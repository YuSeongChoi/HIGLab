# ``HIGHealthKit``

A hands-on tutorial for building a fitness dashboard with HealthKit.

@Metadata {
    @DisplayName("HealthKit Fitness Dashboard")

## Overview

Master the core concepts of HealthKit through a 5-ring growth curriculum.
Learn how to safely read and utilize user health data.

- **Ring 1**: HealthKit Basics & Permission Model
- **Ring 2**: HKHealthStore Setup & Authorization Request
- **Ring 3**: Reading Health Data (Steps, Heart Rate, Sleep)
- **Ring 4**: Workout Records & Statistics Queries
- **Ring 5**: Widgets & Apple Watch Integration
