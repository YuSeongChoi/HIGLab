# ``HIGCloudKit``

A guide to developing cloud-based apps using the CloudKit framework.

## Overview

In this tutorial, you'll build a shared notes app with real-time synchronization using CloudKit, step by step.

CloudKit is Apple's iCloud-based backend service that allows you to implement data storage, synchronization, and sharing features without building your own server.

### What You'll Learn

- CloudKit container and database architecture
- Storing and querying data with CKRecord
- Real-time subscriptions and push notifications
- Collaboration with shared database and CKShare
- Conflict resolution and synchronization strategies
- Using CloudKit Dashboard
