# ``HIGCloudKit``

CloudKit 프레임워크를 활용한 클라우드 기반 앱 개발 가이드입니다.

## Overview

이 튜토리얼에서는 CloudKit을 사용하여 실시간 동기화가 가능한 
공유 메모 앱을 단계별로 만들어봅니다.

CloudKit은 Apple이 제공하는 iCloud 기반 백엔드 서비스로,
별도의 서버 구축 없이 데이터 저장, 동기화, 공유 기능을 구현할 수 있습니다.

### 학습 내용

- CloudKit 컨테이너와 데이터베이스 아키텍처
- CKRecord로 데이터 저장 및 쿼리
- 실시간 구독과 푸시 알림
- 공유 데이터베이스와 CKShare를 통한 협업
- 충돌 해결 및 동기화 전략
- CloudKit Dashboard 활용
