# ``HIGSharePlay``

SharePlay로 만드는 공동 시청 앱 - GroupActivities 프레임워크 완벽 가이드

## Overview

FaceTime 또는 메시지를 통해 원격에 있는 친구들과 함께 미디어를 감상하거나,
앱 내 경험을 실시간으로 공유할 수 있는 SharePlay 기능을 구현합니다.

GroupActivities 프레임워크를 활용해 동기화된 미디어 재생, 커스텀 데이터 전송,
참가자 상태 관리까지 SharePlay의 모든 것을 마스터하세요.
