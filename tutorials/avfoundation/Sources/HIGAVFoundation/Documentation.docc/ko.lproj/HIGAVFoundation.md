## Overview

10단계 커리큘럼으로 AVFoundation의 핵심을 마스터합니다.
사진 촬영부터 비디오 녹화, 실시간 필터까지 카메라 앱의 전 과정을 직접 구현하며 배웁니다.

- **Chapter 1-2**: AVFoundation 아키텍처와 캡처 세션
- **Chapter 3-5**: 사진 촬영, 비디오 녹화, 프리뷰 레이어
- **Chapter 6-7**: 메타데이터 캡처와 오디오 녹음
- **Chapter 8-10**: 비디오 편집, 오디오 세션, 실시간 필터
