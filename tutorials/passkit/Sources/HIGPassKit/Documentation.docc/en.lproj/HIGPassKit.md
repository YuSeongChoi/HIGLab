# ``HIGPassKit``

Building a Membership Card App with PassKit - Complete Guide to Wallet Passes and Apple Pay

## Overview

In this tutorial, you'll learn step by step how to create membership cards using the PassKit framework
and integrate Apple Pay payments.

### Learning Objectives

- **Wallet Pass Creation**: Understanding PKPass structure and pass.json design
- **Pass Signing & Distribution**: Signing with certificates and distributing to users
- **Pass Updates**: Implementing real-time pass updates from server
- **Apple Pay Integration**: From payment request configuration to token processing
- **Advanced Features**: In-App Provisioning, payment sheet customization

### Requirements

- Xcode 15 or later
- iOS 17 or later
- Apple Developer Program membership
- Physical device (simulator doesn't support Apple Pay)
