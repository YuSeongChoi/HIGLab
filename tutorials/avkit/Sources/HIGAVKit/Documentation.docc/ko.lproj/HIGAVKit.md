# ``HIGAVKit``

AVKit 프레임워크를 활용한 비디오 플레이어 구현 가이드입니다.

## Overview

이 튜토리얼에서는 AVKit과 AVFoundation을 활용하여 
전문적인 비디오 플레이어를 구현하는 방법을 단계별로 배웁니다.

AVKit은 iOS 8에서 도입된 프레임워크로, 
시스템 수준의 미디어 재생 UI와 기능을 제공합니다.

### 학습 내용

- AVPlayerViewController를 활용한 비디오 재생
- 재생, 일시정지, 탐색 등 기본 제어
- Picture in Picture(PiP) 지원
- AirPlay를 통한 외부 디스플레이 스트리밍
- 자막 및 오디오 트랙 선택
- 재생 속도 조절
- 챕터 네비게이션과 시간 이동
- 플레이리스트 및 연속 재생
- tvOS에 최적화된 미디어 경험
