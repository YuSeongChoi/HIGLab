# ``HIGAVKit``

A guide to implementing video players using the AVKit framework.

## Overview

In this tutorial, you'll learn step by step how to implement 
a professional video player using AVKit and AVFoundation.

AVKit is a framework introduced in iOS 8 that provides 
system-level media playback UI and features.

### What You'll Learn

- Video playback using AVPlayerViewController
- Basic controls like play, pause, and seeking
- Picture in Picture (PiP) support
- Streaming to external displays via AirPlay
- Subtitle and audio track selection
- Playback speed control
- Chapter navigation and time seeking
- Playlists and continuous playback
- Media experience optimized for tvOS
