# ``HIGSwiftData``

SwiftData로 만드는 할 일 관리 앱 TaskMaster — 완벽 실습 튜토리얼

@Metadata {
    @DisplayName("SwiftData 마스터 클래스")
}

## Overview

10단계 성장고리 커리큘럼으로 SwiftData의 모든 것을 마스터합니다.
Core Data의 복잡함을 벗어나, Swift 네이티브 방식으로 데이터를 다루는 법을 배웁니다.

### 🎯 최종 결과물: TaskMaster

카테고리별 할 일 관리, 우선순위 정렬, iCloud 동기화까지 갖춘 
완성도 높은 Todo 앱을 처음부터 끝까지 만들어봅니다.

### 🏆 학습 목표

- **Ring 1-2**: SwiftData 기초 — @Model, Container, Context
- **Ring 3-4**: 데이터 조회 — @Query, CRUD 연산
- **Ring 5-6**: 고급 모델링 — 관계 설정, 필터링
- **Ring 7-8**: UI 통합 — SwiftUI 연동, 리스트/상세뷰
- **Ring 9-10**: 프로덕션 — 마이그레이션, CloudKit

### ⚡ 핵심 특징

| Core Data | SwiftData |
|-----------|-----------|
| NSManagedObject 상속 | @Model 매크로 |
| .xcdatamodeld 파일 | 순수 Swift 클래스 |
| NSFetchRequest | @Query 프로퍼티 래퍼 |
| NSPredicate (문자열) | #Predicate (타입 안전) |
| 수동 컨텍스트 관리 | 자동 트래킹 |
