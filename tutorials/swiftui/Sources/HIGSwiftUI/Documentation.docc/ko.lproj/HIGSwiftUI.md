# ``HIGSwiftUI``

SwiftUI로 만드는 ChefBook 레시피 앱 - 선언적 UI의 모든 것을 마스터합니다.

@Metadata {
    @DisplayName("ChefBook: SwiftUI 마스터 클래스")
}

## Overview

10단계 실습 커리큘럼으로 SwiftUI의 핵심을 완전 정복합니다.
실제로 동작하는 "ChefBook" 레시피 앱을 만들면서 배웁니다.

이 튜토리얼은 Apple Human Interface Guidelines의 철학을 바탕으로,
**사용자 경험을 최우선**으로 생각하는 앱 개발 방법을 알려드립니다.

### 🍳 무엇을 만드나요?

**ChefBook** - 나만의 레시피를 저장하고 관리하는 앱입니다.

- 레시피 목록 브라우징
- 카테고리별 필터링
- 즐겨찾기 기능
- 요리 타이머
- 단계별 가이드

### 📚 커리큘럼 구성

| 파트 | 주제 | 핵심 개념 |
|-----|------|---------|
| **Part 1** | 기초 다지기 | View, body, 선언적 UI |
| **Part 2** | 레이아웃 | Stack, Grid, 정렬 |
| **Part 3** | 상태 관리 | @State, @Binding |
| **Part 4** | 데이터 표시 | List, ForEach |
| **Part 5** | 네비게이션 | NavigationStack, NavigationLink |
| **Part 6** | 모달과 알림 | Sheet, Alert, Dialog |
| **Part 7** | 컴포넌트화 | 재사용 가능한 뷰 |
| **Part 8** | 애니메이션 | withAnimation, transition |
| **Part 9** | 아키텍처 | Environment, EnvironmentObject |
| **Part 10** | 완성 | 앱 마무리 |

### 💡 학습 방식

각 챕터는 다음 패턴을 따릅니다:

1. **개념 설명** - 왜 이것이 필요한지 이해
2. **코드 예제** - 실제 동작하는 코드 작성
3. **ChefBook 적용** - 우리 앱에 기능 추가
4. **도전 과제** - 스스로 해보기
