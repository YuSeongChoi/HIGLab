## Overview

Completely master SwiftUI fundamentals through a 10-step hands-on curriculum.
Learn by building a working "ChefBook" recipe app.

This tutorial is based on Apple Human Interface Guidelines philosophy,
teaching app development methods that **prioritize user experience**.

**ChefBook** - An app to save and manage your personal recipes.

- Recipe list browsing
- Category-based filtering
- Favorites feature
- Cooking timer
- Step-by-step guide
