# ``HIGTipKit``

Building an Onboarding Guide with TipKit - Displaying App Feature Tips

## Overview

In this tutorial, you'll learn how to naturally guide users
through app features using Apple's TipKit framework.

TipKit is supported from iOS 17 and macOS 14,
allowing fine-grained control over tip display timing, conditions, and frequency.
