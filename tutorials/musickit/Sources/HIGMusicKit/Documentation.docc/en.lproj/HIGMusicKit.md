# ``HIGMusicKit``

A tutorial for developing a music player app using the MusicKit framework.

## Overview

In this tutorial, you'll build a music player app 
integrated with Apple Music using Apple's MusicKit framework.

Learn the entire process of music app development, from 
requesting music permissions to catalog search, playback control, and lyrics display.
