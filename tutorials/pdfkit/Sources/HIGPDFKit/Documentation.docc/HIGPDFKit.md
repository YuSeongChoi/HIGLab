# ``HIGPDFKit``

A guide to implementing a PDF reader using the PDFKit framework.

## Overview

In this tutorial, you'll learn how to build a fully-featured 
PDF reader app step by step using PDFKit.

PDFKit is an Apple framework that provides functionality for displaying, 
searching, annotating, and editing PDF documents on macOS and iOS.

### What You'll Learn

- Rendering PDFs using PDFView
- Loading documents and page navigation
- Text search and highlighting
- Adding annotations with PDFAnnotation
- Implementing a thumbnail grid
- Creating PDF documents
- Editing with page add/delete
