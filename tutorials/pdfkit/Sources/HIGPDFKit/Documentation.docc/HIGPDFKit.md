# ``HIGPDFKit``

PDFKit 프레임워크를 활용한 PDF 리더 구현 가이드입니다.

## Overview

이 튜토리얼에서는 PDFKit을 사용하여 완전한 기능을 갖춘 
PDF 리더 앱을 단계별로 구축하는 방법을 배웁니다.

PDFKit은 macOS와 iOS에서 PDF 문서를 표시하고, 
검색하고, 주석을 달고, 편집하는 기능을 제공하는 Apple 프레임워크입니다.

### 학습 내용

- PDFView를 사용한 PDF 렌더링
- 문서 로드 및 페이지 네비게이션
- 텍스트 검색 및 하이라이트
- PDFAnnotation으로 주석 추가
- 썸네일 그리드 구현
- PDF 문서 생성
- 페이지 추가/삭제 편집
