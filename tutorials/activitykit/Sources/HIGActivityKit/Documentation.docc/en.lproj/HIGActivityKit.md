# ``HIGActivityKit``

Create real-time update experiences using Live Activities and Dynamic Island.

## Overview

In this tutorial, you'll learn the core concepts of ActivityKit by building a delivery tracking app.

- Display real-time information on the lock screen with Live Activity
- Implement compact/expanded layouts in Dynamic Island
- Remote updates using Push Tokens
