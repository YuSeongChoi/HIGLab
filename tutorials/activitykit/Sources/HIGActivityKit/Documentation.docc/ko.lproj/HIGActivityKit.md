# ``HIGActivityKit``

Live Activities와 Dynamic Island를 활용한 실시간 업데이트 경험을 만듭니다.

## Overview

이 튜토리얼에서는 배달 추적 앱을 만들며 ActivityKit의 핵심 개념을 학습합니다.

- Live Activity로 잠금 화면에 실시간 정보 표시
- Dynamic Island에서 컴팩트/확장 레이아웃 구현
- Push Token을 활용한 원격 업데이트
