# ``HIGEventKit``

Learn how to manage iOS calendars and reminders with EventKit.

## Overview

In this tutorial, you'll use Apple's EventKit framework to build 
a fully-featured schedule management app.

We'll cover everything from requesting calendar permissions to creating, 
editing, and deleting events, setting up recurrence rules, 
alarms, and managing reminders.
